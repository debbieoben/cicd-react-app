 # CI/CD React Application

Automated deployment pipeline using AWS CodePipeline, CodeBuild, S3, and CloudFront.

## Features
- Automated builds on Git push
- React production optimization
- CloudFront CDN distribution
- Build info tracking

## Pipeline Flow
GitHub → CodePipeline → CodeBuild → S3 → CloudFront

## Technologies
- React 18
- AWS CodePipeline
- AWS CodeBuild  
- S3 Static Hosting
- CloudFront CDN
